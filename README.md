This repository is my organized collection of everything I learn in **Penetration Testing**.  
It contains:

- Deep explanations of the **OWASP Top 10**
- Custom **OSINT scripts** (Top 10 tools)
- Step-by-step **lab walkthroughs** with notes and commands

The goal is to keep everything clean, searchable, and easy to revisit.
